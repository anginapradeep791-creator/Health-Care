# Project Overview

Healthy Life Hospital is a responsive frontend healthcare website developed using HTML5, CSS3, and JavaScript. The website provides users with information about the hospital, medical departments, doctors, healthcare services, emergency care, appointment booking, gallery, and contact details.

The project is designed with a modern, clean, and user-friendly interface that works smoothly on desktop, tablet, and mobile devices.

---

# Objectives

* Design a responsive healthcare website.
* Apply HTML5 semantic elements.
* Create an attractive interface using CSS3.
* Add basic JavaScript interactivity.
* Implement basic SEO practices.
* Deploy the website using Netlify.
* Host the project on GitHub.
